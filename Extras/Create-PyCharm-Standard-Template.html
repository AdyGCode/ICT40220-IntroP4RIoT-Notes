PyCharm (and in fact all the JetBrains Software) have the ability to create a template that is used on all source files.

We can create them for Python, HTML, and many more languages and frameworks.

This FAQ shows you how to create:

  • A Header file for Python Source Files
  • A Stub File for a Python Script

The Steps

Open Pycharm

Click on the Hamburger Icon

Click on the File menu Item

Click on Settings

Note: To bypass these steps use:

  • CTRL + ALT + S (PC), or
  • CMD + , (Mac)

Click on Editor on the left side of the dialog box, and press &rarrow; to expand the list

Click on File and Code Templates

On the Right side of the Dialog box, now locate the "Python Script" entry, and click on it.

In the editor area that is now shown add:

#parse ("header.py")

    def print_hi(name):
        print(f'Hi, {name}')

if __name__ == '__main__':
    print_hi('Python')


Click APPLY to save the changes.


Now click on the Includes tab

Click on the + sign to create a new include file

In the name on the right side enter header and the extension should be py.

In the editor area add:



"""
ADD APP/SCRIPT/MODULE TITLE

ADD SUMMARY OF APP/SCRIPT/MODULE HERE

------------------------------------------------------------------------------
Project:    ${PROJECT_NAME}
Folder:     ${DIR_PATH}
Filename:   ${FILE_NAME}
Author:     YOUR_NAME <YOUR_EMAIL_ADDRESS>
Version:    0.0
------------------------------------------------------------------------------
"""

Remembering to replace YOUR_NAME and YOUR_EMAIL_ADDRESS with your name and email address, respectively.


Click Apply and then Click OK


Now try creating a new Python File.
